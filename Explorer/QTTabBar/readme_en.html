Requirements

Windows 7, 8, 8.1, 10 (32bit/64bit) + .NET Framework 4.0 or later.

Install

Run the installer - QTTabBar.exe.
In Windows 7, QTTabBar is under View - Toolbars in the Menu Bar. If Menu Bar isn't visible, please set it visible from "Organize" button of the folder band. (figure. 1)
In Windows 8 and 10, you can activate it from drop-down menu of 'Options' button of 'View' tab in the Ribbon. (figure. 2)

For more information, refer to the documents.




Uninstall

Run QTTabBar.exe. Old QTTabBar can be uninstalled by the newest QTTabBar.exe.


Tools

Tools are installed to C:\Program Files\QTTabBar.

QTPopup.exe

Displays Desktop Tool Menu.
To activate Desktop Tool, right click on taskbar and check Toolbars -> QT Tab Desktop Tool.
Desktop Tool provides QTTabBar functionalities to desktop view.
Also shows group/recent tab menus etc. by double-click on taskbar and desktop.
QTPopup.exe commands to show these menus.

Usage
QTPopup.exe [/X:xpos] [/Y:ypos] [/desktopcontents] [/group] [/recenttab] [/applauncher] [/recentfile]
/X x position where the menu is displayed.
/Y y position where the menu is displayed.
ex /X:500 /Y:300
Pops up at mouse cursor without location specified.
/group Displays group menu.
/recenttab Displays recent tab menu.
/applauncher Displays application launcher menu.
/recentfile Displays recent file menu.
/desktopcontents Displays items on the desktop in menu.

/group, /recenttab, /applauncher, /recentfile, /desktopcontents can be used alone.


RestoreWindow.exe

Restores tabs contained by a previous window.
Usage
RestoreWindow.exe [/w]
/w Restores tabs in new folder window.


Distributor.exe

Edits, downloads or uploads Language file, and downloads or uploads skin image files.
Usage
Distributor.exe [/L] [/I]
(no arguments) Runs Language File Editor.
/L Runs Language File Downloader.
/I Runs Image File Downloader.

Language File Editor
Old language files need to be converted with Language File Editor. First "Load from file..." and then "Save As..".
